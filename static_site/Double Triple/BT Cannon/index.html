BT Cannon Setup
The BT Cannon is a Double Triple setup variant that uses a different stacking approach compared to DT Cannon. The BT (Back-to-Back) Cannon emphasizes consecutive T-Spin Triple execution with optimal piece usage.
BT Cannon Pattern
The BT Cannon creates a structure optimized for back-to-back T-Spin Triples. This pattern maximizes scoring potential through consecutive clears.
Key Differences
The BT Cannon differs from DT Cannon in its stacking approach and piece requirements. Understanding these differences helps in choosing the right pattern for each situation.
